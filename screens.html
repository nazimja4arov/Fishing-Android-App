App Screens Overview

What each screen does in the Android app

Home Tutorial Screens

Home Screen

Entry point of the app. Shows navigation options like “Add Fish” and “Leaderboard”. Implemented as a Compose @Composable with buttons that call your navigation function using the sealed Route class.

Add Fish Screen

Form where the user enters species, weight, location, and maybe a short note. On save, it sends data to a ViewModel which updates the list of catches and, indirectly, the leaderboard.

Leaderboard Screen

Displays a ranking of users by total points (or weight of fish). Uses a LazyColumn in Compose to show rows with rank, user name, and points. Sorting is handled in ViewModel/repository, UI just reads the state.

Tutorial Screen (planned)

Future screen you can add to explain how the app works directly inside the app. It can reuse parts of this website’s text and be reachable from the Home Screen via a new navigation route.
